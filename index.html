Acerca de este blog:

Este blog está hecho con la finalidad de presentar una página en donde se presenten diferentes estilos, se presente la inserción de material visual y audio visual cómo vídeos, imágenes o música. Este blog está creado principalmente para la matería de Programación Web impartida por la docente María Angélica Mártinez Herrera.
Las herramientas utilizadas para la creación de este blog fueron solamente Visual Studio Code; en él se codificó toda la estructura de las páginas que contiene el blog, incluyendo esté párrafo.
Se emplearon hojas de estilo para darle características visuales a los elementos de todas las páginas; los estilos están redactados en su propio archivo.

Finalmente, todos los archivos (o al menos la mayoría) utilizados para el blog fueron subidos a un repositorio en GitHub, donde se van actulizando cada que hay alguna modificación.
